# План разработки Stroop-теста

## Обзор проекта
Stroop-тест — интерактивное веб-приложение для оценки когнитивного контроля. Пользователь видит слово, обозначающее цвет, написанное цветным шрифтом, и должен выбрать цвет шрифта, игнорируя семантическое значение слова.

**Технологический стек:**
- Frontend: React 19 с TypeScript
- Сборка: Vite
- Стилизация: CSS Modules
- Тестирование: Vitest + React Testing Library
- Хранение состояния: React Context + useReducer

**Текущее состояние:** проект инициализирован с базовым шаблоном Vite + React, есть файлы `src/App.tsx`, `src/main.tsx`, `package.json` с зависимостями React 19. Структура папок соответствует спецификации частично (отсутствуют папки `components/`, `context/`, `hooks/`, `utils/`, `types/`).

## Цели разработки
1. Реализовать все основные экраны (приветствие, тестирование, результаты)
2. Внедрить генерацию стимулов с конгруэнтностью
3. Реализовать три уровня сложности с разными настройками
4. Добавить расчёт метрик и сохранение истории
5. Обеспечить адаптивный и визуально приятный UI
6. Написать тесты для ключевых функций

- **Сложность генерации стимулов:** тщательно протестировать алгоритм на всех уровнях сложности.
- **Производительность анимаций:** использовать CSS-анимации вместо JavaScript.
- **Совместимость браузеров:** проверить на основных браузерах (Chrome, Firefox, Safari).
- **Объём состояния:** использовать useReducer для эффективного управления.

## Критерии успеха
- Приложение работает без ошибок в соответствии со спецификацией.
- Пользователь может пройти тест на любом уровне сложности.
- Метрики рассчитываются корректно и сохраняются.
- UI адаптивен и интуитивно понятен.
- Код покрыт тестами и хорошо документирован.
